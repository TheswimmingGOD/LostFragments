# Lost Fragments

Lost Fragments is a Fabric mod for Minecraft Java Edition 26.1.2. It adds an
amethyst infusion table that gives tools and armor utility abilities without
changing their base stats.

## Development setup

For setup instructions, please see the [Fabric Documentation page](https://docs.fabricmc.net/develop/getting-started/creating-a-project#setting-up) related to the IDE that you are using.
